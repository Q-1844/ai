# Agnes-2.0-Flash 生成的游戏代码

这些是由 Agnes-2.0-Flash 模型生成的太空射击游戏代码。

## 运行方法

```bash
pip install pygame
python game_01.py
```

## 代码统计

| 文件 | 行数 | 说明 |

| game_01.py | 1080 | 使用Enum+dataclass，结构完整 |

| game_03.py | 1059 | 使用dataclass |
| game_04.py | 1060 | 最完整架构 |

| game_06.py | 975 | 基础版本 |

## 注意事项

- 所有代码都使用程序化图形（无需外部图片）
- 音效为占位符或合成音
- 需要安装 pygame 库
